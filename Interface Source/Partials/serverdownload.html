Download and install WeebIRC on your computer.

WeebIRC requires a server to run on your local network. This server will do all the heavy work for you, like parsing all the anime information from different resources (Myanimelist, Nibl) and download/stream them. It's absolutely essential that you read the tutorial to ensure that everything works like it should!

Linux

  • Step 1

    WeebIRC is written in .NET, which can only run on linux if mono-project has been installed. The following steps will show you how to install mono-project using the linux terminal!

    Ubuntu
    linux@user:$ sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 3FA7E0328081BFF6A14DA29AA6A19B38D3D831EF
    linux@user:$ echo "deb http://download.mono-project.com/repo/debian wheezy main" | sudo tee /etc/apt/sources.list.d/mono-xamarin.list
    linux@user:$ sudo apt-get update
    linux@user:$ sudo apt-get install mono-complete


    CentOS 7, Fedora 19 (and later), and derivatives

    yum install yum-utils
    rpm --import "http://keyserver.ubuntu.com/pks/lookup?op=get&search=0x3FA7E0328081BFF6A14DA29AA6A19B38D3D831EF"
    yum-config-manager --add-repo http://download.mono-project.com/repo/centos/
    sudo yum install mono-complete

  • Step 2

    Navigate to the directory/location where you put the server and unpack the tar file you just downloaded.

    linux@user:$ cd /path/to/directory/where/weebircserver/resides
    linux@user:/path/to/directory/where/weebircserver/resides$ tar -xvf WeebIRCServerLinux.tar

  • Step 3

    Start the server. The application will ask you if it should output logdata in the console and/or if it should write to a log file. It will also ask if it should download mkvextract, which you need to be able to view subtitles inside the browser!


    linux@user:/path/to/directory/where/weebirc/resides$ sudo mono WeebIRCServerLinux.exe

  • Done!
